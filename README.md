# Address Book

## Client Ask

> We want a single page application that replaces our rolodex that is currently handled by Jane at the front desk. 
>
> It only needs to track:
>   - First Name
>   - Last Name
>   - Physical Address

## Developer Responses

- Login
- Data Transfer
- Database
- Search Bar 
  - First Name
  - Last Name
  - Address
- Home page
- Update addresses
- Colors: {Black. Lime Green. Light Grey}

## Technology

- Database: MongoDB via Atlas
- Application: React
- Hosting: <None>
